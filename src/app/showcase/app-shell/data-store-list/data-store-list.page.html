DataStore - List example

When you fetch data from a remote API, sometimes you request a list of items. If that’s the case you would end up having an Observable of type Observable<Array<YourModel>>.

The DataStore class works by assigning a shell property to any object. 

const myObj = new YourModel();
Object.assign(myObj, {isShell: false});

Object.assign([1, 2, 3], {isShell: false});

In Javascript, everything is an object (including Arrays). That’s why you can assign properties to Arrays independently of their values.

In this example we keep track of the current state of the data stream in a local data property.

Note how we define its type using Typescript intersection typing like this data: Array<YourModel> & ShellModel; to indicate it's an array that has a shell model appended (by the DataStore).
