DataStore - Multiple DataStores example

Sometimes your view displays different types of data, each of them with a different model and data source.

If this is your use case, then you can use multiple data stores to fetch each data stream.

Note: This use case is different from the Combined Data Sources because in this case you have two data streams that will change independently one from the other updating different parts of the view without causing a full view 're-loading'.
